# 实验报告：采样、混叠与自适应下采样

## 一、实验目的

1. 理解图像下采样过程中混叠（aliasing）现象的产生原因。  
2. 掌握使用高斯低通滤波进行抗混叠下采样的方法。  
3. 探究不同高斯滤波参数 σ 对混叠抑制和图像模糊的影响，验证理论最优公式。  
4. 实现基于局部梯度分析的自适应下采样，并与全局统一下采样进行质量对比。

## 二、实验原理

### 1 混叠现象
当图像的下采样倍数不足时，高于新奈奎斯特频率的高频成分会“折叠”到低频区域，产生虚假的低频条纹，称为混叠。典型表现如棋盘格缩小后出现的摩尔纹。

### 2 抗混叠滤波
在下采样前先对图像进行低通滤波，滤除超出新采样率的高频成分，可有效抑制混叠。高斯滤波是一种常用的抗混叠滤波器，其频响特性由标准差 σ 控制。σ 越大，滤波越强，高频衰减越多。

### 3 最优 σ 理论公式
对于 M 倍下采样（缩小为 1/M），理论最优抗混叠滤波的 σ 约为：
σ_opt =0.45×M
本实验中固定 M=4，故理论最优 σ = 1.8。

### 4 自适应下采样
图像不同区域的高频成分不同：平坦区域变化缓慢，纹理或边缘区域变化剧烈。使用统一的 σ 会导致平坦区过度模糊或纹理区混叠残留。自适应下采样根据局部梯度估计高频能量，对高频区域用较大 σ 滤波，低频区域用小 σ 滤波，从而在整体上获得更好的视觉效果和更低的误差。

## 三、实验环境与工具

- 操作系统：Windows 11 + WSL 2 (Ubuntu 22.04)  
- 编程语言：Python 3.10  
- 主要库：OpenCV 4.13.0 (headless 无界面版)、NumPy、Matplotlib  
- 测试图像：自生成的棋盘格（512×512，格子大小16）和径向频率扫描图（chirp，512×512，k=4）

## 四、实验内容与结果

### 1 第一部分：混叠观察与抗混叠效果

####（1） 棋盘格实验
- 原图：清晰棋盘格，高频丰富（图 `checker.png`）。
- 直接下采样（1/4）：出现明显摩尔纹（混叠），频谱中高频成分折叠到低频区域（图 `checker_direct.png` 和 `checker_spectrum_direct.png`）。
- 高斯滤波后下采样（σ=1.8）：混叠消失，图像变柔和，频谱高频被有效衰减（图 `checker_gaussian.png` 和 `checker_spectrum_gaussian.png`）。

#### （2）Chirp 图实验
- 原图：频率从中心向外逐渐升高，边缘处密集（图 `chirp.png`）。
- 直接下采样：外围高频区域出现严重混叠，形成混乱的条纹（图 `chirp_direct.png`）；频谱可见高频成分折叠（图 `chirp_spectrum_direct.png`）。
- 高斯滤波后下采样：混叠基本消除，图像模糊，频谱高频被抑制（图 `chirp_gaussian.png` 和 `chirp_spectrum_gaussian.png`）。

**结论**：高斯低通滤波能有效抑制下采样引起的混叠，但会损失部分高频细节。

---

### 2 第二部分：不同 σ 值对混叠抑制的影响

固定 M=4（缩小1/4），对棋盘格分别用 σ=0.5, 1.0, 2.0, 4.0 进行高斯滤波后下采样，结果如下（对应图 `checker_sigma_0.5.png` 等）：

- **σ=0.5**：滤波太弱，混叠依然明显，摩尔纹残留。
- **σ=1.0**：混叠部分减少，但仍可见轻微摩尔纹。
- **σ=2.0**：混叠完全消失，图像稍模糊。
- **σ=4.0**：混叠消失，但图像过度模糊，细节丢失严重。

**最优 σ 判断**：σ=2.0 附近效果最佳（混叠消失且模糊度可接受）。理论最优 σ=1.8 与实际观察一致。

---

### 3 第三部分：自适应下采样与全局统一下采样对比

采用径向 chirp 图作为测试对象：
- 全局统一下采样（σ=1.8）：产生图 `uniform_down.png`，平坦区模糊，纹理区混叠较少。
- 自适应下采样（分块大小32×32）：产生图 `adaptive_down.png`，平坦区较清晰，纹理区混叠抑制效果好。

#### 误差图对比
误差图 = |直接下采样 - 恢复下采样| × 4（图 `error_uniform.png` 和 `error_adaptive.png`）：
- 统一 σ 的误差图：亮区较多，尤其在边缘附近误差大。
- 自适应 σ 的误差图：整体更暗，说明误差更小，尤其在平坦区域几乎无误差。

**结论**：自适应下采样能根据局部内容调整滤波强度，在保持平坦区清晰度的同时有效抑制纹理区的混叠，整体误差低于全局统一下采样。

## 五、实验总结

本实验通过生成棋盘格和 chirp 测试图，直观演示了图像下采样中的混叠现象及其频域表现。验证了高斯低通滤波的抗混叠作用，并确定了固定下采样倍数 M=4 时最优 σ 约为 1.8~2.0，与理论值 1.8 吻合。进一步实现了基于梯度分析的自适应下采样，相比全局统一下采样，自适应方法在主观质量和客观误差上均表现更优。
